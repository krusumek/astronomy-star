# 🚀 ตะลุยอวกาศ: กำเนิดและวิวัฒนาการของดาวฤกษ์

สื่อการสอนแบบเกม simulation สำหรับนักเรียนชั้นมัธยมศึกษาปีที่ 4 (ฟิสิกส์/โลกดาราศาสตร์)
พัฒนาโดย **นายสุเมฆ สุทธิบริบาล** ครูชำนาญการ กลุ่มสาระวิทยาศาสตร์ฯ โรงเรียนสตึก สพม.บุรีรัมย์

▶️ **เล่นเกม:** https://krusumek.github.io/astronomy-star/

## เนื้อหา
ผู้เล่นจะได้ปั้นดาวฤกษ์ด้วยมือตัวเอง ตั้งแต่เกิดจนดับ ผ่าน 5 ด่าน พร้อมเก็บการ์ดสะสมดาว:

| ด่าน | เนื้อหา |
|-----|---------|
| 1. กำเนิดดาว | แรงโน้มถ่วงยุบตัวของเนบิวลาเป็นโปรโตสตาร์ |
| 2. จุดติดดาว | ปฏิกิริยาฟิวชัน + สมดุลอุทกสถิต |
| 3. สีและอุณหภูมิ | ความสัมพันธ์สี ↔ อุณหภูมิ (K) ของดาวฤกษ์จริง |
| 4. จุดจบของดาว | 4 ชะตากรรมตามมวลตั้งต้น (แคระน้ำตาล/แคระขาว/นิวตรอน/หลุมดำ) |
| 5. แผนภาพ H–R (เสริม) | จัดกลุ่มดาวตามอุณหภูมิและโชติมาตรสัมบูรณ์ |

จบเกมมีแบบทดสอบท้ายบท 10 ข้อ

## ไฟล์ในโปรเจกต์
- `index.html` — ตัวเกม (ไฟล์ HTML เดี่ยว ไม่มี dependency ภายนอกยกเว้นภาพ/เพลงที่โหลดจากอินเทอร์เน็ต)
- `00_แผนสื่อ_ตะลุยอวกาศ.md` — แผนการออกแบบสื่อและบันทึกการพัฒนาแต่ละเวอร์ชัน
- `01_Content_Blueprint.md` — เนื้อหาวิชาการ บทพูดตัวละคร ข้อสอบ และแหล่งอ้างอิง

## เทคนิคที่ใช้
HTML + CSS + Canvas 2D + Web Audio API (ไม่ใช้เฟรมเวิร์ก) เก็บความคืบหน้าผู้เล่นด้วย `localStorage`
