- < 0.08 → ดาวแคระน้ำตาล · 0.08–8 → ดาวแคระขาว · 8–25 → ดาวนิวตรอน · > 25 → หลุมดำ

# ด่าน 1 · กำเนิดดาว (Star Formation)

**จุดประสงค์การเรียนรู้:** อธิบายได้ว่าดาวฤกษ์เกิดจากเนบิวลา (เมฆแก๊ส-ฝุ่น) ที่ยุบตัวด้วยแรงโน้มถ่วง

**ข้อเท็จจริงหลัก:**
- ดาวฤกษ์เกิดในเนบิวลา (nebula) = กลุ่มแก๊ส (ส่วนใหญ่ไฮโดรเจน) และฝุ่น
- เมื่อบริเวณหนึ่งมีมวล/ความหนาแน่นมากพอ แรงโน้มถ่วงดึงสสารยุบรวมเข้าหากัน
- ยิ่งยุบ → แกนกลางยิ่งร้อนและหนาแน่นขึ้น → เกิดเป็น **ดาวฤกษ์ก่อนเกิด (protostar)**
- ยิ่งสะสมมวลมาก แรงโน้มถ่วงยิ่งแรง การยุบยิ่งเร็ว

**กลไกเกม:** ลากนิ้วกวาดก้อนแก๊ส/ฝุ่นมารวมกัน → มาตรวัด "มวลสะสม" เพิ่มขึ้น → ถึงเกณฑ์ → เมฆเรืองแสงยุบตัวเป็นโปรโตสตาร์
**เกณฑ์ผ่าน:** สะสมมวลถึง 100% จนจุดกำเนิดโปรโตสตาร์

**บทน้องโนวา:**
- เปิดด่าน: "ยินดีต้อนรับนักสำรวจ! ตรงหน้าเราคือ *เนบิวลา* แหล่งกำเนิดดาว ลองกวาดแก๊สกับฝุ่นมารวมกันสิ — แรงโน้มถ่วงจะทำงานเอง!"
- ระหว่างเล่น: "เยี่ยม! ยิ่งมวลมาก แรงโน้มถ่วงยิ่งดึงแรงขึ้นนะ"
- hint (ถ้าติด): "ลองลากแก๊สที่อยู่ไกล ๆ เข้ามากองตรงกลางเพิ่มอีกนิด ให้มวลถึงเกณฑ์"
- ผ่านด่าน: "เกิดดาวฤกษ์ก่อนเกิดแล้ว! นี่คือ *โปรโตสตาร์* แต่มันยัง 'ไม่ติดไฟ' จริง ๆ ไปจุดไฟกันต่อ!"

# ด่าน 2 · จุดติดดาว (Ignition: Fusion + Hydrostatic Equilibrium)

**จุดประสงค์การเรียนรู้:** อธิบายการเกิดปฏิกิริยาเทอร์โมนิวเคลียร์ฟิวชัน และสมดุลอุทกสถิตที่ทำให้ดาวคงรูป

**ข้อเท็จจริงหลัก:**
- โปรโตสตาร์ยุบตัวจนแกนกลางร้อนถึง ~**10 ล้านเคลวิน** → เริ่มปฏิกิริยา **ฟิวชัน (fusion)**
  *(แกนดวงอาทิตย์จริง ~15 ล้านเคลวิน)*
- ฟิวชัน: ไฮโดรเจน (H) 4 นิวเคลียส รวมเป็นฮีเลียม (He) 1 นิวเคลียส + ปลดปล่อยพลังงานมหาศาล
- ถ้ามวลโปรโตสตาร์ **< 0.08 M☉** แกนกลางร้อนไม่พอ → ฟิวชันไม่เกิด → กลายเป็น **ดาวแคระน้ำตาล (brown dwarf)** "ดาวที่ล้มเหลว"
- **สมดุลอุทกสถิต (hydrostatic equilibrium):** ดาวคงรูปอยู่ได้เพราะ
  - แรงโน้มถ่วง (ดึงเข้าด้านใน) = ความดันจากพลังงานฟิวชัน (ดันออกด้านนอก)
  - ถ้าฟิวชันน้อยไป → แรงโน้มถ่วงชนะ → ดาวยุบ
  - ถ้าฟิวชันมากไป → ความดันชนะ → ดาวพองตัว

**กลไกเกม (3 เฟส):**
- 2.1 เลื่อนสไลเดอร์เพิ่มอุณหภูมิแกนจนถึง 10–15 ล้านเคลวิน
- 2.2 animation: H 4 ตัว วิ่งมารวมเป็น He 1 ตัว + แสงพลังงาน
- 2.3 ภารกิจสมดุล: ปรับ "อัตราฟิวชัน" ให้แรง 2 ฝั่งสมดุล คุมเข็มอยู่โซนเขียวให้ครบเวลา
**เกณฑ์ผ่าน:** คุมสมดุลอยู่โซนเขียวครบเวลาที่กำหนด (เช่น 8 วินาที)

**บทน้องโนวา:**
- เปิดด่าน: "โปรโตสตาร์ต้องร้อนถึงราว 10 ล้านเคลวินถึงจะ 'จุดติด' ได้ ลองเพิ่มอุณหภูมิแกนดูสิ!"
- หลัง 2.2: "นี่แหละฟิวชัน! ไฮโดรเจน 4 ตัวรวมเป็นฮีเลียม แล้วปล่อยพลังงานออกมา — ดาวสว่างขึ้นแล้ว!"
- เปิด 2.3: "ทีนี้มาเป็นภารกิจสุดหิน — ดาวจะอยู่ได้ต้อง *สมดุล* ระหว่างแรงโน้มถ่วงที่ดึงเข้า กับความดันฟิวชันที่ดันออก คุมให้อยู่ตรงกลางนะ!"
- hint: "ฟิวชันแรงไป ดาวจะพอง / น้อยไป ดาวจะยุบ ค่อย ๆ ปรับทีละนิด"
- ผ่านด่าน: "สุดยอด! ดาวเข้าสู่ *สมดุลอุทกสถิต* แล้ว มันจะอยู่แบบนี้ไปอีกหลายล้าน–พันล้านปี ช่วงนี้เรียกว่า *แถบลำดับหลัก*"

# ด่าน 3 · แถบลำดับหลัก (Main Sequence: มวล–สี–ความสว่าง–อายุ)

**จุดประสงค์การเรียนรู้:** อธิบายความสัมพันธ์ระหว่างมวลดาวกับสี (อุณหภูมิ) ความสว่าง (โชติมาตร) และอายุขัย

**ข้อเท็จจริงหลัก:**
- **มวลคือตัวกำหนดทุกอย่าง** ของดาวบนแถบลำดับหลัก
- **สีบอกอุณหภูมิ** (สเปกตรัม OBAFGKM — ร้อน→เย็น):

| ชนิด | สี | อุณหภูมิผิว (K) |
|------|----|----|
| O | น้ำเงิน | > 30,000 |
| B | น้ำเงิน-ขาว | 10,000–30,000 |
| A | ขาว | 7,500–10,000 |
| F | ขาว-เหลือง | 6,000–7,500 |
| G (ดวงอาทิตย์) | เหลือง | 5,200–6,000 (~5,800) |
| K | ส้ม | 3,700–5,200 |
| M | แดง | 2,400–3,700 |

- **โชติมาตร (magnitude) — สเกลกลับด้าน:** ตัวเลขยิ่งน้อย/ติดลบ = ยิ่งสว่าง
  - ต่างกัน 5 โชติมาตร = สว่างต่างกัน 100 เท่า
  - ตัวอย่าง: ดวงอาทิตย์ −26.7 · ดวงจันทร์เต็มดวง −12.7 · ดาวซิริอุส −1.46 · ดาวเหนือ +1.98 · ตาเปล่ามองเห็นจาง ~+6
- **มวลกับอายุ:** มวลมาก = ร้อน-สว่าง-สีน้ำเงิน-เผาเชื้อเพลิงเร็ว = **อายุสั้น** / มวลน้อย = เย็น-หรี่-สีแดง = **อายุยืนมาก**

**กลไกเกม:** สไลเดอร์ "มวลดาว" ตัวเดียว เลื่อนแล้วเห็นสี/ความสว่าง/โชติมาตร/อายุ เปลี่ยนแบบ real-time
**ทดสอบก่อนผ่านด่าน (2 แบบ):**
1. จับคู่การ์ดดาว (เช่น สีน้ำเงิน ↔ ร้อนที่สุด, โชติมาตร −1 ↔ สว่างกว่า +5, เรียงลำดับความสว่างจากโชติมาตร)
2. ตั้งค่าดาวตามโจทย์ (เช่น "สร้างดาวสีน้ำเงินมวลมาก", "สร้างดาวที่อายุยืนที่สุด")

**บทน้องโนวา:**
- เปิดด่าน: "ตอนนี้ดาวเราอยู่ใน *แถบลำดับหลัก* แล้ว ลองเลื่อนมวลดูสิ — เห็นไหมว่าทุกอย่างเปลี่ยนตามมวลหมดเลย!"
- เรื่องสี: "ดาวสีน้ำเงินร้อนกว่าดาวสีแดงนะ! สวนทางกับก๊อกน้ำที่บ้านเลย 😆"
- เรื่องโชติมาตร: "ระวังนะ! *โชติมาตร* ยิ่งตัวเลขน้อยหรือติดลบ = ยิ่งสว่าง เช่น ดวงอาทิตย์ −26.7 สว่างสุด ๆ ส่วนดาวจาง ๆ จะเป็น +5, +6"
- เรื่องอายุ: "ดาวมวลมากเท่ติดไฟแรง แต่ใช้เชื้อเพลิงเปลือง เลยอายุสั้น ส่วนดาวมวลน้อยอยู่ได้นานเป็นหมื่นล้านปี!"
- ผ่านด่าน: "เก่งมาก! เข้าใจแล้วว่ามวลกำหนดทุกอย่าง ทีนี้มาดูกันว่าดาวแต่ละมวล *จบชีวิต* ยังไง"

# ด่าน 4 · จุดจบของดาว (Stellar Death by Mass)

**จุดประสงค์การเรียนรู้:** จำแนกจุดจบของดาวฤกษ์ตามมวลเริ่มต้น 4 กรณี

**ข้อเท็จจริงหลัก (เกณฑ์มวล — รอตรวจตามหมายเหตุด้านบน):**

| มวล protostar | เส้นทางวิวัฒนาการ → จุดจบ |
|---------------|--------------------------|
| **< 0.08 M☉** | แกนร้อนไม่พอ ฟิวชันไม่เกิด → **ดาวแคระน้ำตาล (brown dwarf)** |
| **0.08 – 8 M☉** | แถบลำดับหลัก → ดาวยักษ์แดง (red giant) → เนบิวลาดาวเคราะห์ (planetary nebula) → **ดาวแคระขาว (white dwarf)** |
| **8 – 25 M☉** | แถบลำดับหลัก → ดาวยักษ์ใหญ่แดง (red supergiant) → **ซูเปอร์โนวา (supernova)** → **ดาวนิวตรอน (neutron star)** |
| **> 25 M☉** | แถบลำดับหลัก → ดาวยักษ์ใหญ่แดง → **ซูเปอร์โนวา** → **หลุมดำ (black hole)** |

- ดาวแคระขาวคงรูปด้วย "ความดันดีเจนเนอเรซีของอิเล็กตรอน"
- ดาวมวลมากจบด้วยการระเบิดซูเปอร์โนวา แล้วแกนยุบเป็นดาวนิวตรอน (มวลปานกลาง) หรือหลุมดำ (มวลมากสุด)

**กลไกเกม:** สไลเดอร์มวล + ปุ่ม "เร่งเวลา" → ดู animation วิวัฒนาการจนถึงจุดจบของกรณีนั้น
**เกณฑ์ผ่าน:** ทดลองครบทั้ง 4 กรณี (ปลดล็อกการ์ดสะสมจุดจบทั้ง 4 แบบ)

**บทน้องโนวา:**
- เปิดด่าน: "ถึงเวลาความจริง — มวลตอนเกิดจะกำหนดว่าดาวจะ *ตาย* แบบไหน ลองเลื่อนมวลแล้วกดเร่งเวลาดูสิ!"
- กรณี <0.08: "มวลน้อยเกินไป ฟิวชันไม่เคยเกิดเลย มันเลยเป็นแค่ *ดาวแคระน้ำตาล* — ดาวที่ล้มเหลว"
- กรณี 0.08–9: "ดาวแบบดวงอาทิตย์เรา! พองเป็นยักษ์แดง สลัดเปลือกเป็นเนบิวลาสวย ๆ เหลือแกนเป็น *ดาวแคระขาว*"
- กรณี 9–25: "ตูม! *ซูเปอร์โนวา*! แกนยุบแน่นจนเป็น *ดาวนิวตรอน* — ช้อนเดียวหนักเป็นพันล้านตัน!"
- กรณี >25: "มวลมหาศาล! ระเบิดแล้วยุบจนแสงหนีไม่ออก กลายเป็น *หลุมดำ* 🕳️"
- ผ่านด่าน: "ครบทั้ง 4 ชะตากรรมแล้ว! นี่คือวงจรชีวิตของดาว — เกิดจากเนบิวลา แล้วบางส่วนก็กลายเป็นเนบิวลาใหม่ให้ดาวรุ่นต่อไป"

# ด่าน 5 · แผนภาพ H–R (เสริม / ท้าทาย)

**จุดประสงค์การเรียนรู้:** อ่านและจัดตำแหน่งดาวบนแผนภาพ Hertzsprung–Russell

**ข้อเท็จจริงหลัก:**
- แกนนอน (x) = อุณหภูมิ/สี — **ร้อน(น้ำเงิน)อยู่ซ้าย → เย็น(แดง)อยู่ขวา** (ค่าลดจากซ้ายไปขวา ตามธรรมเนียม H–R)
- แกนตั้ง (y) = ความสว่าง/โชติมาตรสัมบูรณ์ — สว่างมากอยู่บน
- **แถบลำดับหลัก (main sequence):** แนวทแยงจากบนซ้าย (ร้อน-สว่าง) → ล่างขวา (เย็น-หรี่) — ดาว ~90% รวมทั้งดวงอาทิตย์อยู่แถบนี้
- **ดาวยักษ์/ยักษ์ใหญ่:** อยู่บนขวา (เย็นแต่สว่างเพราะใหญ่)
- **ดาวแคระขาว:** อยู่ล่างซ้าย (ร้อนแต่หรี่เพราะเล็ก)

**กลไกเกม:** น้องโนวายื่นการ์ดดาว (บอกสี+ความสว่าง) → ลากวางตำแหน่งบนแผนภาพ → ระบุว่าเป็นดาวกลุ่มไหน
**เกณฑ์ผ่าน:** วางถูกโซน + ระบุกลุ่มถูก ครบจำนวนการ์ด

**บทน้องโนวา:**
- เปิดด่าน: "ด่านท้าทายรวมยอด! แผนภาพ H–R คือแผนที่ของดาวทั้งจักรวาล เอาทุกอย่างที่เรียนมาใช้เลย!"
- ผ่านด่าน: "ระดับนักดาราศาสตร์ตัวจริง! 🌟 จบภารกิจตะลุยอวกาศอย่างสมบูรณ์แบบ"
